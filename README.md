# Virtusa Mini Projects

A collection of mini projects developed during training, using Java, Python, and SQL.

---

## Projects

### 1. Library Management System (Core Java)

* Manage books and users
* Issue and return books
* Fine calculation
* Built using Java + MySQL

### 2. Weather Data Analyzer (Python)

* Analyze weather data
* Generate basic graphs
* Built using Python (Pandas, Matplotlib)

### 3. Retail Sales Analysis (SQL)

* Database design and normalization
* SQL queries for insights
* Built using MySQL

---

## How to Run

### Clone Repo

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### Library Management System

1. Open in IntelliJ
2. Setup MySQL
3. Update DB credentials
4. Run main file

### Weather Analyzer

```bash
pip install pandas matplotlib
python main.py
```

### SQL Project

1. Open MySQL Workbench
2. Run schema.sql
3. Run data.sql
4. Execute queries
